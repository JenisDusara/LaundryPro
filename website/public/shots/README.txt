PUT YOUR REAL SOFTWARE SCREENSHOTS HERE
=======================================

Drop PNG (or JPG) screenshots of YOUR actual LaundryMax software into this
folder, using EXACTLY these file names. The website picks them up automatically
— no code change needed. Until a file exists, a clean placeholder frame shows.

Required files:

1. about.png        -> Big dashboard used in the "About" + Hero sections.
                       Best: a wide desktop dashboard screenshot (16:10).

2. store-web.png    -> "Key Features · For Stores" (the Store Web App).
                       A desktop/browser dashboard screenshot (16:10).

3. mobile-app.png   -> "Store Mobile App" section.
                       A single phone screen screenshot (tall, ~9:19).

4. admin.png        -> "Key Features · For Admin".
                       The admin panel / analytics screenshot (16:10).

5. why-choose.png   -> "Why Choose LaundryMax?" section image.
                       Any product or dashboard screenshot (roughly square/4:3).

Tips:
- PNG preferred. Keep each under ~1.5 MB if possible.
- Wide/desktop shots look best around 1600x1000 px.
- Phone shot: capture just the app screen (no device frame) — the site adds
  the phone frame for you.
- After adding files, just refresh the site (restart dev server if needed).
